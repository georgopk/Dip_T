Instalação

Siga as seguintes instruções depois de instalar o Matlab ou Octave.

Windows

  • Faça o Download da última versão 64bit openEMS. (Versões antigas podem ser encontradas aqui: win32, win64)
  • Extraia (Unzip) o arquivo em uma pasta de sua escolha e.g. "D:/" (o arquivo zip possue uma pasta chamada openEMS)
  • Acrescente openEMS ao path do matlab:
    • Usando o matlab crie um arquivo chamado startup.m com o seguinte código: 
      addpath('D:/openEMS/matlab');
    • Guarde o arquivo na pasta de inicio do Matlab. Para saber qual é a pasta de inicio execute no Matlab: 
      which startupsav

o resultado deve ser algo do tipo:

 >> C:\MATLABR11\toolbox\local\startupsav.m

sendo assim, guarde o arquivo startup.m na pasta C:\MATLABR11\toolbox\local\

  • Pronto! Verifique sua instalação seguindo os passos do First Steps Tutorial
  • De uma olhada na seção de software adicional.
  • Pode comentar sua experiência no openEMS-Forum. (em inglês)

Linux

A forma recomendada é instalando desde o código fonte do openEMS. (em inglês)

Software Adicional

Software adicional que pode ser útil:

Paraview

"Paraview é uma aplicação livre e multi-plataforma para a visualização e o analise de dados." Paraview Homepage

Paraview pode ser usado para visualizar arquivos vtk que contenham os depósitos de campo (field dumps) gerados por openEMS. No Linux, você pode usar seu administrador de pacotes para instalar o Paraview. Para outros sistemas operativos, faça o download em : https://www.paraview.org/paraview/resources/software.html
