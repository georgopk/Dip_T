openSUSE

Repositories:

  • openSUSE 13.2: https://download.opensuse.org/repositories/science/openSUSE_13.2
  • openSUSE Leap 42.1: https://download.opensuse.org/repositories/science/openSUSE_Leap_42.1
  • openSUSE Tumbleweed contains openEMS in main OSS repository.

Use YaST to add the repository and then install openEMS, AppCSXCAD and octave-openEMS or openEMS-matlab.

Alternatively use zypper (e.g. openSUSE Leap 42.1):

zypper ar -f http://download.opensuse.org/repositories/science/openSUSE_Leap_42.1/ science
zypper install openEMS AppCSXCAD
zypper install <octave-openEMS or openEMS-matlab>

For use with Octave (will install Octave):

zypper install octave-openEMS

Or with Matlab (will not install Matlab):

zypper install openEMS-matlab
