Main window of AppCSXCAD

AppCSXCAD is a visualization program, capable of showing the structure defined inside a CSXCAD xml-file. It is a thin wrapper around the QCSXCAD library.

Features:

  • 2D view
  • 3D view
  • selectively display properties
  • Export the CSXCAD file into:
    • povray
    • STL
    • X3D
    • Polydata-VTK
    • PNG
