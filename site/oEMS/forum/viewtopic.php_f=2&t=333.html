Yes! I just installed paraview from brew and ran the first Parallel Plate Waveguide tutorial. I had installed openEMS in a Ubuntu virtual machine last week so I understood how it was supposed to work.

After running the brew script, I did have to create an opt directory in my home folder

Code: Select all

mkdir ~/opt
then create a symbolic link

Code: Select all

ln -s /usr/local/Cellar/openEMS/HEAD ~/opt/openEMS
after that I was able to start Octave and add the paths in the Octave command line with these commands:

Code: Select all

addpath('~/opt/openEMS/share/openEMS/matlab');
addpath('~/opt/openEMS/share/CSXCAD/matlab');
savepath()
