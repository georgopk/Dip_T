Hi,

compiling openEMS on Linux is very easy and even our recommended way of using openEMS. Doing this on Windows is rather difficult because of all the necessary open-source libs we use (e.g. hdf5, boost, vtk etc.). To compile these using mingw is not very easy...

The easiest way to do this is maybe use my already compiled libs and follow these steps:
  • Download the Qt SDK (using mingw)
  • Get the pre-compiled libs needed by openEMS: http://openems.de/download/win32/openEM ... s_win32.7z
  • Get the openEMS and CSXCAD sources, either from openEMS.de or the latest tree from https://github.com/thliebig (put them in the same path as the libs above)
  • Use Qt-Creator to compile CSXCAD and openEMS
If you want to contribute to openEMS, I highly recommend doing it on Linux!

regards
Thorsten
