Hi,

to me it sounds like everything is installed properly...

Just open a tutorial example with the Matlab editor in D:\FDTD\openEMS\matlab\Tutorials and run it (e.g. by pressing F5)

The Matlab script will create an xml file with all necessary information and call RunOpenEMS to run the binary program openEMS using this file.
