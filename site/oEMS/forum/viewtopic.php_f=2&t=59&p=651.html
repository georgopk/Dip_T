Hi Seb,

that was my first though as well, but I checked, they use different class names...

I think that he compiled (the right) tinyxml without stl support:

in tinyxml.h:

Code: Select all

 #ifndef TINYXML_INCLUDED
 #define TINYXML_INCLUDED
 
+#ifndef TIXML_USE_STL
+	#define TIXML_USE_STL
+#endif
+
 #ifdef _MSC_VER
^^ That was a changes I needed to introduce in tinyxml, the official packages do contain excatly this lines, but not the original source-code...
