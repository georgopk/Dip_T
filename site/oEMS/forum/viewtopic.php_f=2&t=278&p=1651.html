Thorsten, I fixed...

I edited the AppCSXCAD.sh script to change the DYLD_LIBRARY_PATH (Mac OS X MATLAB uses DYLD_LIBRARY_PATH instead of LD_LIBRARY_PATH):

Code: Select all

export DYLD_LIBRARY_PATH=/System/Library/Frameworks/ImageIO.framework/Versions/\
A/Resources/:/usr/local/lib/:$DYLD_LIBRARY_PATH
Of note, the home brew installed libraries (including the libtiff problem libraries reside in /usr/local/lib as symlinks to /usr/local/Cellar:
