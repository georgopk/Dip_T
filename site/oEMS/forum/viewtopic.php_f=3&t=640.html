Hi Paraview is indeed very powerful and thus not easy to use. But at least it can do nearly everything you could ever want
But we usually have no other option than using Paraview.
Why is that? You can access all data from within Matoab/Octave, thus you can plot/draw in whatever (limited) way Octave/Matlab is capable...
You just need to dump in the hdf5 file format instead of vtk.
If you still want to use Paraview in these cases there a matlab files than can read and convert hdf5 to vtk.
I usually do 3D dumps in hdf5 only and extract planes with these scripts when needed...
I'm pretty sure that all the tools and script you need are there.

regards
