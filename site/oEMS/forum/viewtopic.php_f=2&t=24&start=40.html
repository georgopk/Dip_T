Hi Meiyor,

you are working on Mac OS X? openEMS is not tested on that platform. Did you manage to compile an openEMS version?

Anyway I think your Mac is recognised as a unix system and the linux shell script for openEMS is executed. It should be included if you checked out the git-version of openEMS (it is not in the win32 version).
But I don't know if this shell script is working on a Mac.

Maybe you have a look at the "invoke_openEMS.m" in the openEMS/matlab folder

Code: Select all

if isunix
	openEMS_Path = [openEMS_Path 'openEMS.sh'];
else
	openEMS_Path = [openEMS_Path 'openEMS'];
end
