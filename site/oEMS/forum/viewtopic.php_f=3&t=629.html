hellow
i an simulating on cruved waveguide and i have much noise coming from the metal-air adge .I want to add some absorbing material over there to lower the noise (I want something like the the MUR boundary condition) what is the right way to do it? or do someone have other idea?
thanks

my code:

Code: Select all

function [ output_args ] = prb_sim1( input_args )
clear all
close all
clc
run=0;
physical_constants;
unit=1e-6 ;
wg_inner_r_min=1.5;
wg_inner_r_max=2;
wg_tick=0.4;
wg_len = 5 %the aim job is to set the wave propagation direction to be the same as the waveguide tangent direction
aim_len=1; 
metal_tick=0.5;
cox_len=0.5;  
n=1.5;
f_0=3.84e14;
NumItr=500
cox_inner_r = wg_inner_r_min + (wg_inner_r_max-wg_inner_r_min)*((-aim_len-wg_len/2)/(wg_len/2))^2
lambda0=c0/f_0/unit;
mesh_res=lambda0./[10 10 10]
min_val=mesh_res(3);
movment=1;

Sim_Path='tmp1';
Sim_CSX ='coax1_wg.xml';

FDTD= InitFDTD('NrTS',NumItr,'OverSampling',5);

BC={'MUR' 'MUR' 'MUR' 'MUR' 'MUR' 'MUR'};
FDTD=SetBoundaryCond(FDTD,BC);

FDTD=SetSinusExcite(FDTD,f_0);

CSX=InitCSX();
mesh.x=SmoothMeshLines([0 (2*(cox_inner_r+wg_tick+metal_tick))],mesh_res(1));
mesh.y=SmoothMeshLines([0 (2*(cox_inner_r+wg_tick+metal_tick))],mesh_res(2));
mesh.z=SmoothMeshLines([-(cox_len+aim_len+mesh_res(3)*movment) (wg_len)],mesh_res(3));
%mesh_res(3)=mesh.z(2)-mesh.z(1); %some time the mesh is not what you expect it to be...
CSX=DefineRectGrid(CSX,unit,mesh);

sim_size=size(mesh.x,2)*size(mesh.y,2)*size(mesh.z,2)

CSX = AddMetal(CSX,'metal');
CSX = AddMaterial( CSX, 'my_n' );

CSX = SetMaterialProperty( CSX, 'my_n', 'Epsilon', n^2);

%placeing the excitetion coax

start_cox = [mesh.x(end)/2,mesh.y(end)/2,mesh.z(movment)];
stop_cox  = [mesh.x(end)/2,mesh.y(end)/2,start_cox(3)+cox_len];
r_in=cox_inner_r;
r_o=r_in+wg_tick;
r_os=r_o+metal_tick;

[CSX,port{1}] = AddCoaxialPort( CSX, 10, 1, 'metal', 'my_n', start_cox, stop_cox, 'z',r_in ,r_o ,r_os ,'ExciteAmp', 1 );

%crating the aim coax
start_aim = [mesh.x(end)/2,mesh.y(end)/2,stop_cox(3)] ;  %the aim starts where the coax end
stop_aim  = [mesh.x(end)/2,mesh.y(end)/2,start_aim(3)+aim_len] ;  %and it's length is aim len
middel_x=mesh.x(end)/2;
middel_y=mesh.y(end)/2;
%adding the matal cylinder to aim the wave to the wg insident angle
CSX=AddBox(CSX, 'metal', 0, [mesh.x(1),mesh.y(1),start_aim(3)], [mesh.x(end),mesh.y(end),stop_aim(3)]);

for i=[start_aim(3):min(mesh_res(3),min_val):stop_aim(3)]
        inner_r =  wg_inner_r_min + (wg_inner_r_max-wg_inner_r_min)*((-wg_len/2+i)/(wg_len/2))^2;
        
        %addig the matarial shell -drilling inside the matal
        CSX = AddCylindricalShell(CSX, 'my_n', 10, [middel_x,middel_y,i], [middel_x,middel_y,i+min(mesh_res(3),min_val)], inner_r+0.5*wg_tick, wg_tick);
        %warning if the mesh in z anix is too low it couas an errow in the
        %radius since the mesh in z anix is diffrent then mes_res
        
end

for i=[stop_aim(3):min(mesh_res(3),10*min_val):stop_aim(3)+wg_len]
        inner_r =  wg_inner_r_min + (wg_inner_r_max-wg_inner_r_min)*((-wg_len/2+i)/(wg_len/2))^2;
        
        %addig the matarial shell -drilling inside the matal
        CSX = AddCylindricalShell(CSX, 'my_n', 10, [middel_x,middel_y,i], [middel_x,middel_y,i+min(mesh_res(3),min_val)], inner_r+0.5*wg_tick, wg_tick);
end

CSX = AddDump(CSX,'Et');
CSX = AddBox(CSX,'Et',0,[mesh.x(1) mesh.y(1) mesh.z(1)],[mesh.x(end) mesh.y(end) mesh.z(end)]);    


mkdir(Sim_Path);
WriteOpenEMS([Sim_Path '/' Sim_CSX],FDTD,CSX);
CSXGeomPlot([Sim_Path '/' Sim_CSX]);
if run==1 
    RunOpenEMS(Sim_Path , Sim_CSX,'');
end
end

Hi,

I did not have a closer look at your script, but in general there are no special absorbing materials inside the domain like for the borders (MUR/PML).
What you can always do is create some areas with losses like absorbers at areas where you want to dampen unwanted fields. This would be pretty much like you would do it in the real world too.
