Post by sebastian » Thu 02 Feb 2012, 21:58

It should be easy to use openEMS to solve such a structure.
Read the tutorials at http://openems.de/index.php/Tutorials and look at the provided examples in the source code archive.
The provided examples to start with are Coax_CylinderCoords.m and Coax_Cylindrical_MG.m
Both model a coaxial cable. Use the methods described in http://openems.de/index.php/Tutorial:_C ... rn_Antenna to add a antenna pattern calculation to your model.

But if you have to write the fdtd code, then have a look at our source code. Be careful: obey the GPL license and give proper credit.

Take some time and read http://openems.de/index.php/OpenEMS#Installation.
Then read http://openems.de/index.php/Tutorial:_First_Steps.
This should answer questions 1 and 2.

For question 4, download the source code and have a look. It's an open source project, so you have the oppertunity to have a detailed look how this all works.
