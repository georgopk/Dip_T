Post by chinsoon » Fri 02 Dec 2016, 05:38

Ok I managed to fix the "octave-gui.exe has stopped working" problem by following the instructions given here:
http://savannah.gnu.org/bugs/?46702

which says:
in the Octave command line type in: graphics_toolkit,
then switch the default toolkit to gnuplot:
graphics_toolkit('gnuplot')

After this fix, Octave could generate pop-up windows with graphical plots.

But the "AppCSXCAD.exe has stopped working" still exists after that. So still need a fix for "AppCSXCAD.exe has stopped working".
