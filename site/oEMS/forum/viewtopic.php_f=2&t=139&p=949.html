Hi,

have you added both path to CSXCAD and openEMS?

Code: Select all

If you have installed from a linux package repository, running setup is not necessary (and not possible since it would need root privileges). Just try running e.g. the Simple_Patch_Antenna tutorial to see if everything is working fine...

Post by thorsten » Tue 04 Jun 2013, 11:23

Did you install the AppCSXCAD package?

To run the examples/tutorials you should open them, but first do "save as" to somewhere else in your home directory... otherwise you cannot modify or run them as normal user...
Or just copy every example/tutorial to your home directory first...

I don't know why "h5readatt_octave" was not already build in your openEMS package, but doing it as root is not nice but seems to work.
But other than for that, you really should not run openEMS as root user!!

There is always the possibility to just compile openEMS etc. from source, it is not that difficult and you don't have all this trouble
http://openems.de/index.php/Compile_from_Source

You could even try the latest dev-sources, if you replace:

Code: Select all

git clone git://openEMS.de/$mod.git
by

Code: Select all

git://github.com/thliebig/$mod.git
in the instructions...

regards
Thorsten
